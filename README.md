# Full-Stack Playground

A collection of independent projects spanning **web**, **mobile**, **full-stack**, and **systems** work. Each folder is a self-contained app or exercise with its own README, dependencies, and run instructions.

## Projects

| Folder | Summary | Stack (high level) |
|--------|---------|---------------------|
| [Image-Lab-Mobile](./Image-Lab-Mobile) | Expo app with Firebase auth, Firestore, camera flows, and image workflows | Expo, React Native, Firebase |
| [Uber-Clone-App-HandsOn](./Uber-Clone-App-HandsOn) | Ride-booking style mobile UI (onboarding, auth screens, tabs) | Expo, React Native, NativeWind |
| [perceptron-labs](./perceptron-labs) | Reddit-style simulation ideas forum with votes and real-time-style sync via Redis | Next.js, Prisma, SQLite, Redis, Tailwind |
| [Portfolio](./Portfolio) | Personal portfolio site | React, Tailwind, Framer Motion |
| [systems-portfolio](./systems-portfolio) | Systems-focused portfolio (distributed systems emphasis), deployed with Vite | React, Vite, MUI, Radix, Tailwind |
| [SwiggyUI](./SwiggyUI) | Front-end clone exercise modeled on food-delivery UI | React, Parcel |
| [Visulizations](./Visulizations) | Data file upload, parsing, and visualization experiments (D3) | React, D3 |
| [user-login](./user-login) | Phone number and OTP step-through UI (client-side flow demo) | React (CRA) |
| [Image-Crawler](./Image-Crawler) | Multi-threaded Java servlet crawler: images from a URL and same-domain subpages | Java, Maven, Jetty |
| [snapchat_custom_filter](./snapchat_custom_filter) | Placeholder for Snapchat / Lens Studio filter notes or assets | — |

## Getting started

Open any project folder and follow its README. There is no single install at the repo root.
