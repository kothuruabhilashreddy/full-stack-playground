# Swiggy UI Clone

Front-end **clone / tribute** UI inspired by the **Swiggy** food-delivery web experience: layout, components, and styling practice without calling real Swiggy APIs.

## Stack

- **React** 19
- **Parcel** 2 (bundler and dev server)

## Run

```bash
npm install
npm start
```

Parcel serves `index.html` by default (see terminal output for the local URL).

## Build

```bash
npm run build
```

Output is produced by Parcel for static hosting.

## Testing

```bash
npm test
```

(Jest is configured in `package.json`.)
