# Visualizations

React app for **uploading data files**, **parsing** structured content (including XML-style data), and **driving visualization** experiments with **D3**. It includes scaffolding for **attribute classification** (numerical vs categorical) alongside chart components.

## Features (high level)

- **File upload** pipeline with parsing helpers (`FileUpload`)
- **D3** for charting (see components under `src/Components/`)
- **Home** view composes upload, classifier, and chart placeholders

## Run

```bash
npm install
npm start
```

Runs the CRA dev server (default [http://localhost:3000](http://localhost:3000)).

## Stack

- React (Create React App)
- D3
